SOBRE

Sou um desenvolvedor web nascido em Piracicaba, São Paulo.

Gosto de projetos que me permitam aprender, aplicar padrões e conhecer coisas novas na área de desenvolvimento web. Atualmente apaixonado pelo desenvolvimento de aplicações SPA, usando AngularJS 1.X e ASP.NET Web Api como backend. Este conjunto me possibilitou adquirir muitos conhecimentos em javascript, estrutura de projetos e C# propriamente.

Minha carreira no mundo do desenvolvimento teve início com o uso de Web Forms, em 2010, enquanto desenvolvia soluções e prestava manutenções em sistemas da área de transporte coletivo público. Tive diversas aventuras no uso da IDE do Visual Studio, visto que foi meu primeiro contato profissional com a ferramenta e com Orientação a Objetos na prática.

Após adquirir os conhecimentos iniciei uma nova aventura, usando PHP para desenvolver soluções para a área de saúde pública, iniciando um uso mais abrangente de Javascript com jQuery para diversas funcionalidades que dependendiam altamente em ajax para respostas rápidas.

Sentindo-me confiante com estas tecnologias, procurei ampliar meus conhecimentos durante o desenvolvimento do meu TCC em 2012, voltando para o mundo de tecnologias da Microsoft, usando ASP.NET MVC 4. Usando meu conhecimento acadêmico, experiências e com ajuda do Google para as pesquisas de tutoriais e artigos, o projeto foi um sucesso, atingindo as expectativas de desenvolvimento e conclusão do projeto.

Foi uma grande aventura, com grandes resultados e conhecimentos adquiridos. PHP continuava divertido, mas C# era mais.

Em 2013 tive uma oportunidade de emprego onde voltei a atuar com ASP.NET, mas dessa vez usando o ASP.NET MVC 5 com Bootstrap como framwork css. Foi a primeira vez que tive o contato com um design padronizado e templates prontos - até então trabalhava diretamente com CSS/HTML próprios de cada empresa.

Em 2015 iniciei meus trabalhos com Web Api e AngularJS para criar aplicações híbridas usando o Ionic 1 como ferramenta de manipulação para gerar os arquivos para o dispositivo desejado.

Procuro me aperfeiçoar nas tecnologias AngularJS e Web Api para fornecer soluções mais robustas, mas sem esquecer do uso de MVC onde é necessário e aplicável.
